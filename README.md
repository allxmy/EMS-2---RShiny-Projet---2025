## Information

Le fichier SAE_DPE contient tout les fichiers nécessaire pour le bon fonctionnement de l'application R. Voici son contenu :
1. app.R : contient tout le code de l’application Shiny (UI + serveur).
2. www : dossier où sont stockées les ressources.
3. renv : dossier contenant l’environnement isolé avec les versions locales des packages.
4. renv.lock : liste exacte des packages et versions nécessaires pour reproduire l’environnement.
5. rsconnect : mémorise les paramètres de déploiement pour shinyapps.io.
6. manifest.json : décrit la configuration technique envoyée lors du déploiement sur shinyapps.io.

Les 4 derniers fichiers ont été créer pour le bon déploiement de l'application sur shinyapps.io
